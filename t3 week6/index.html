Build Scene (First Version)

This is the scene I built at the beginning. I floated some daily objects in the village on the water. I added Mesh Collider and Rigidbody, and needed to cancel gravity and add the previous floating code. At the same time, I also added Fog to make it look more mysterious.

Then I started to try to add a controllable player. At first, I wrote the code to control it myself. WASD controlled forward, backward, left and right, Q and E controlled sink and lift. I found that the effect was not good, and the movement was not smooth enough. And I set the player to be a fishing boat, so when he dived into the water, it felt a little strange.

So I set the control code to the third-person perspective in Unity Asset Store, and set the player to a submarine/spaceship shape, which looks very different from the local environment, so it fits the doomsday story I want.

Final Scene

Then I changed the skybox to make it look like there were a lot of exhaust gas and tall buildings in the distance, and the sky showed the color of doomsday.

To make the dystopia feel stronger, I also added some giant moving spiders that crawl on land and in water.
